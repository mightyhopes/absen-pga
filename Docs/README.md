# Dokumentasi Perbaikan Sistem Absensi & Payroll

Folder `Docs` disusun untuk memisahkan dokumentasi sistem aktif (terbaru) dengan rekam jejak historis perbaikan bug.

## Struktur Folder

1. **`AUDIT-ENGINE.md`**
   - Isi: Penjelasan cara kerja dan format output untuk Engine 2 (Mode Audit Internal).
   - Status: **Aktif**. Dokumentasi ini masih relevan dengan kode terbaru.

2. **`Fix_Bug_Jaman_Dulu/` (Folder Arsip)**
   - Isi: File historis analisis (`01_...`), perbaikan v2 (`02_...`), dan finalisasi v3 (`03_...`) dari masa-masa awal sistem direstrukturisasi (sekitar bulan Mei-Juni awal).
   - Status: **Arsip (Deprecated)**. Disimpan hanya untuk keperluan melacak sejarah logika kode dan asal mula perombakan algoritma lembur. Tidak lagi mencerminkan logika 100% kode `Kode.gs` saat ini, namun berguna jika ingin mengerti kronologi penyesuaian aturan absen.

## Keterangan Tambahan
Jika Anda baru pertama kali melihat sistem ini, baca `README.md` utama di direktori luar untuk memahami gambaran besar sistem.
